## 📂 Projektstruktur
- `index.html`: Startseite mit dem aktuellen Saisonkalender 2026.
- `nav.html` & `footer.html`: Zentrale Bausteine für ein einheitliches Design.
- `style.css`: Das gesamte visuelle Design (Tennis-Grün Look).
- `script.js`: Lädt die Navigation und den Footer dynamisch in alle Unterseiten.
- `/turniere`: Unterseiten für Eltern-Kind-Turnier, Davis-Cup, Champions-Race etc.

## 🛠 Features
- **Dynamische Navigation:** Änderungen am Menü müssen nur in einer Datei (`nav.html`) vorgenommen werden.
- **Responsive Design:** Die Turniertabellen und das Layout passen sich an Smartphones an.
- **Rechtssicher:** Inklusive Vorlagen für Impressum und Datenschutz.
